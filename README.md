 **Status do Projeto:** Em desenvolvimento

O **Campus Rota** é um aplicativo de impacto social desenvolvido para auxiliar estudantes universitários de baixa renda ou em situação de vulnerabilidade socioeconômica. O foco principal são alunos que possuem bolsa de estudos integral (100%) ou que comprovadamente não têm condições financeiras para arcar com custos de transporte privado (como vans universitárias).


### Como funciona:
* **Definição de Trajeto (Ponto X ao Ponto Y):** O estudante insere o seu ponto de partida e o seu destino (por exemplo: do *Centro* até a *Anhanguera*). Com isso, o aplicativo calcula ou direciona o usuário para as melhores opções disponíveis de deslocamento.
* **Critério de Acesso:** Possui um sistema de triagem/cadastro para garantir que o benefício e as informações de rotas otimizadas cheguem estritamente a quem mais precisa, conectando estudantes necessitados que fazem trajetos parecidos.


## 🛠️ #2 Linguagem, Pacotes e Requisitos

Atualmente, o **Campus Rota** está disponível em formato de **plataforma Web (Site)** para garantir o acesso mais rápido e fácil para todos os estudantes, sem necessidade de instalação inicial. O suporte nativo para celulares (Android e iOS) está planejado para as próximas etapas do projeto.

### 🌐 Tecnologias e Linguagens Atuais (Web):
* **Frontend (Interface do Usuário):** HTML5, CSS3 e JavaScript / TypeScript
* **Backend (Regras de Negócio e API):** Python / Node.js
* **Banco de Dados:** SQLite3 / PostgreSQL


### 💻 Requisitos do Sistema para Uso:
* **Navegadores Suportados:** Qualquer navegador moderno (Google Chrome, Safari, Microsoft Edge, Firefox, Opera) tanto no computador quanto no navegador do celular.
